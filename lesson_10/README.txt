ДЗ 8. AsyncIO
Для выполнения задания я выбрал другое API (https://api.apilayer.com/) т.к. API
рекомендуемое в примере, считаю непригодным для выполнения задания, по причине
ограничения запросов (5 шт в минуту).
Для того, чтобы добавить или исключить валюту участвующую в запросах на получение
курса валют, достаточно добавить её в список - CURRENCY.
Все запросы выполняются асинхронно, с помощью библиотек AsyncIO & HTTPX.
